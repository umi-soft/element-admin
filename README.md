## Getting started

```bash
# clone the project
git clone https://github.com/umi-soft/element-admin.git

# install dependency
npm install

# develop
npm run dev
```

## Build

```bash
# build for test environment
npm run build:sit

# build for production environment
npm run build:prod
```
